Projeto de Customer Experience Analytics

Pipeline Completo: Python ETL • SQL (Supabase) • Tableau

Sobre o Projeto

Este painel apresenta a análise de NPS (Net Promoter Score) e Satisfação do Cliente de uma empresa fictícia. Os dados foram gerados via Python, processados em banco de dados SQL na nuvem e visualizados neste dashboard interativo.

Tecnologias usadas: Python (Pandas, SQLAlchemy), PostgreSQL (Supabase), Tableau Public e HTML/CSS.
